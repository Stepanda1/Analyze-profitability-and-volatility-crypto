# Анализ доходности и волатильности криптовалют

Учебный проект по дисциплине **«Анализ данных на Python»**. В работе сравниваются доходность, риск и взаимосвязи восьми популярных криптовалют за 365 дней по данным CoinGecko API.

**Авторы:** Козлов Степан, Шембек Александр  
**Период данных:** 15 июня 2025 — 14 июня 2026

## Цель исследования

Сравнить поведение BTC, ETH, BNB, SOL, XRP, DOGE, ADA и TRX по следующим характеристикам:

- накопленная и дневная доходность;
- годовая и 30-дневная скользящая волатильность;
- максимальная просадка;
- отношение доходности к риску;
- корреляция дневных доходностей.

## Исследовательские гипотезы

1. BTC и ETH менее волатильны, чем большинство альткоинов.
2. Более высокая доходность связана с более высоким риском.
3. Доходности криптовалют положительно коррелируют между собой.
4. DOGE более нестабилен, чем крупные инфраструктурные криптовалюты.

## Этапы проекта

```text
Гипотезы → Сбор данных → Очистка → Создание признаков → Анализ → Визуализация → Выводы
```

### 1. Сбор данных

Данные получаются через endpoint `market_chart` CoinGecko API. Для каждой криптовалюты загружаются:

- цена в USD;
- рыночная капитализация;
- общий объём торгов;
- временная метка наблюдения.

### 2. Очистка и предобработка

- преобразование даты к типу `datetime`;
- приведение числовых столбцов к корректным типам;
- сортировка по криптовалюте и дате;
- удаление повторных комбинаций `symbol + date`;
- проверка и предусмотренное заполнение пропусков;
- контроль количества наблюдений по каждой криптовалюте.

После удаления повторных дат осталось **2920 наблюдений: 8 криптовалют × 365 дней**.

### 3. Созданные признаки

- дневная доходность;
- накопленная доходность;
- 30-дневная скользящая волатильность;
- годовая волатильность;
- максимальная просадка;
- упрощённое отношение доходности к риску.

### 4. Анализ и визуализация

В ноутбуке представлены:

- нормализованная динамика цен;
- сравнение накопленной доходности;
- сравнение годовой волатильности;
- динамика 30-дневной волатильности;
- диаграмма «риск — доходность»;
- корреляционная матрица дневных доходностей.

## Основные результаты

| Криптовалюта | Накопленная доходность | Годовая волатильность | Максимальная просадка |
|---|---:|---:|---:|
| TRX | 17,06% | 28,50% | −26,49% |
| BNB | −5,29% | 53,58% | −56,40% |
| ETH | −33,84% | 67,89% | −67,52% |
| BTC | −38,81% | 43,00% | −51,22% |
| XRP | −46,41% | 67,98% | −69,26% |
| DOGE | −51,05% | 79,44% | −71,90% |
| SOL | −52,78% | 72,03% | −74,89% |
| ADA | −72,54% | 77,81% | −83,73% |

Главные выводы для выбранного периода:

- положительную накопленную доходность показал только TRX;
- TRX имел наиболее выгодное сочетание доходности и волатильности;
- DOGE оказался самым волатильным активом;
- ADA показала худшую накопленную доходность и максимальную просадку;
- высокий риск не гарантировал высокую доходность;
- корреляции дневных доходностей в основном положительные.

## Запуск

1. Клонировать репозиторий:

```bash
git clone https://github.com/Stepanda1/-.git
cd -
```

2. Создать виртуальное окружение и установить зависимости:

```bash
python -m venv .venv
```

Windows:

```bash
.venv\Scripts\activate
```

Linux/macOS:

```bash
source .venv/bin/activate
```

```bash
pip install -r requirements.txt
```

3. Создать `.env` по образцу `.env.example` и добавить Demo API key CoinGecko:

```text
COINGECKO_API_KEY=your_api_key_here
```

4. Запустить ноутбук:

```bash
jupyter notebook crypto_analysis.ipynb
```

При отсутствии API-ключа ноутбук может использовать локальный файл `raw_crypto_data.csv`, расположенный рядом с ним.

## Формат локального датасета


## Ограничения

- Рассматривается только один год, поэтому выводы нельзя автоматически переносить на другие периоды.
- Исследование описывает исторические данные и не прогнозирует будущие цены.
- `return_to_risk` является упрощённым показателем и не заменяет коэффициент Шарпа.
- Последний день может быть неполным, если данные загружались до завершения суток.
- Результаты не являются инвестиционной рекомендацией.
