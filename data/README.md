# Локальные данные

Основной источник проекта — CoinGecko API. Если API-ключ недоступен, рядом с ноутбуком можно разместить файл `raw_crypto_data.csv`.

Обязательные столбцы:

```text
date,symbol,coin_id,price,market_cap,total_volume
```

Файл с данными не хранится в репозитории, чтобы не дублировать данные внешнего источника и не увеличивать размер проекта.
